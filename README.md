# 📈 Apple Stock Price Analysis and Prediction

A Data Science project focused on analyzing historical Apple (AAPL) stock market data using Python, data visualization, and machine learning techniques. This project explores stock price trends, market behavior, and predictive analytics using real-world financial data.

## 📌 Project Overview

The objective of this project is to analyze historical Apple stock data and uncover meaningful insights through:

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization
- Trend Analysis
- Machine Learning-based Stock Price Prediction

The project demonstrates how Data Science can be applied to financial datasets for informed decision-making and predictive analytics.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/prithivirathan22/apple-stock-price-analysis
```

Navigate to the project directory:

```bash
cd stock-data-science-project
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python src/main.py
```
